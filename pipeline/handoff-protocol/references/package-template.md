# Handoff Package Template

## 5-Component Package

### 1. One-Page Briefing (from discovery-call-prep)
- Recipient profile (≤80 words)
- MEDDPICC snapshot (8 slots)
- 9 discovery questions (3 per genre)
- 3 pre-staged objections
- 1–3 competitors with counter-positioning
- Call agenda (≤3 bullets)

### 2. MEDDPICC Slot Snapshot
- 8 slots: populated vs unknown
- Per-slot provenance tag
- Highlights which slots the receiving rep should probe

### 3. Conversation History
- Chronological: all Touches sent + all replies received
- Includes: channel, date, framework used, reply classification
- Serves as context continuity for the receiving rep

### 4. Pre-Staged Objection Responses
- 3 most-likely objections with response variants
- Pulled from objection-handling-library
- Framework attribution per variant

### 5. SAL Acceptance Checklist
- 4 criteria from sal-criteria.md
- Pre-checked: ✓ or ✗ per criterion
- Space for rep signature/confirmation

## Delivery Format

- Single document or linked artifacts
- Deliverable via: Slack DM, email, dashboard alert
- Must include accept/reject mechanism (buttons or reply-keyword)
