# Call Script Frameworks & Templates

## 9-Second Opener (≤30 words)

Pattern: **greet by name → state cold-call honestly → 5-second purpose → permission ask**

### Template

```
"Hi {name} — this is {rep} from {company}. This is a cold call — mind if I take 30 seconds? I noticed {specific trigger} — typically {pain pattern}. Curious how you're handling {area}."
```

### Opener Angles (Bosworth)

- **Pain** — "You're hitting X problem" (from ICP P-T-O)
- **Power** — "You have the authority to fix Y"
- **Vision** — "You'd recognize the Z state"

Pick per recipient based on signals + role.

### Cliché Blocklist (Hard Ban)

- "Just calling to check in"
- "I noticed your company" (without specific verified detail)
- "We help X do Y" (generic opener)
- "Following up on our chat" (when there was no chat)

### No-Pretend Rule

Zero phrases implying prior conversation that didn't happen:
- ❌ "As we discussed last week..."
- ❌ "Following up on our conversation..."
- ❌ "Reaching back out about..."

## Discovery Questions (Bosworth + Sandler Pain Funnel)

4–6 questions following: **pain → impact → buying-vision → urgency → consequence-of-inaction**

### Template

```
1. How is the {area} team handling {process} these days?           [Open/curiosity]
2. Where do {role} get stuck most — {aspect A} or {aspect B}?      [Quantify pain]
3. If {metric} were {improved state}, what would that change for you in {quarter}?
4. What have you tried so far?                                      [Current approach]
5. What's keeping that from working?                                [Barrier]
6. Is fixing this a priority in the next {timeframe}, or further out? [Urgency]
```

## Gatekeeper Handler (3 Patterns)

### Pattern A: Specific-Pain Reply

```
"It's about {specific pain area} — I work with {role} at {ICP segment}.
{name}'s name came up because {specific trigger}."
```

### Pattern B: Honest Cold-Call Reply

```
"Honest answer: it's a cold call. I'm trying to reach {name} for a
30-second permission check. Is {he/she} the right person for {area}?"
```

### Pattern C: Permission-Ask

```
"Happy to send an email if that's better — what's the best address?"
```

**NEVER**: pretend prior conversation; use vague buzzwords ("strategic value"); lie about referral source.

## Voicemail Drop (≤15 seconds / ≤45 words)

### Template

```
"Hi {name}, {rep} from {company}. Calling because {specific trigger referencing hook}.
{1-sentence pain observation}. Curious how you're handling {area}.
Call back at {number} — {repeat number digit-by-digit}. Thanks {name}."
```

### Rules

- ≤45 words at natural pace = ~14 seconds
- Name + company (3s) + specific reason (8s) + callback number x2 (4s)
- Callback number repeated twice
- Conversational tone, lower-pitched, slower pace
- Reference specific hook from data-enrichment OR use honest cold-call pattern
