# Profile Template — Discovery Call Prep

## Recipient Profile (≤80 words)

Structure:
- **Name** [verified: data-enrichment or reply-text]
- **Title** [verified: data-enrichment or linkedin]
- **Company** [verified: data-enrichment]
- **ICP Tier + Score** [verified: lead-scoring]
- **Hook** — the verified signal that triggered outreach (e.g., "VP CX hire 2026-04-19 [verified: news URL]")
- **Most recent reply** — 1-line summary of the reply that booked the meeting [verified: reply-text]

## Anti-Fabrication Rule

Every named entity in the profile must carry provenance. "Esme mentioned X" requires the reply text saying X. Never invent context about the recipient.
