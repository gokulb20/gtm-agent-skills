# Bowling-Pin Segment Scoring Rubric

## Origin

The bowling-alley / bowling-pin metaphor is from Geoffrey Moore's *Inside the Tornado* (1995). Segments are not equal — a beachhead win supplies reference momentum to the next segment.

The 5-dimension 25-point rubric below is a simplified house-built scorecard inspired by Moore's logic (target customer characterization — compelling reason to buy, decision-maker accessibility, whole-product feasibility, partner/ally presence, competitive density).

## Scoring Dimensions

| Dimension | What it measures | Score guide |
|---|---|---|
| Pain severity | How acute is the problem for this segment | 1=nice-to-have → 5=blocks growth/breaks workflow |
| Urgency / triggers | Is there an event making it "now" | 0=no trigger → 5=active trigger event in last 90 days |
| Reachability | Can the user reach this buyer with available motion | 1=cold-only → 5=warm intros/active community |
| Reference value | Will winning this segment unlock the next | 1=nobody-cares → 5=unlocks a segment |
| Economic capacity | Can they pay | 1=below floor → 5=priority budget category |

Sum → 5–25 score. Highest = **head pin (beachhead)**.

## Tier Classification

- **Beachhead:** Highest score — full sales motion
- **Pin 2 / Pin 3:** Reachable via beachhead reference power
- **Defer:** Scores too low for current stage or motion

## For Deeper Sequencing

For whole-product gaps, partner ecosystems, and full target-customer characterization, apply Moore's complete framework separately.
