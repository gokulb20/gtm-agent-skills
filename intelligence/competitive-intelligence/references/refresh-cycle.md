# Battle-Card Refresh Cycle

## Trigger Table

| Trigger | Action | SLA |
|---|---|---|
| Quarterly (calendar) | Full review of all top-3 competitor cards | 1 week |
| Score ≥10 signal on a competitor | Touch up affected sections | 1 week |
| Score ≥16 signal | Full card refresh | 2 weeks |
| Win/loss pattern shift | Review wedge sections immediately | 1 week |
| New competitor enters Direct tier | Build new card; review tiering | 2 weeks |

## Anti-Patterns

- **Annual refresh:** Too slow. Quarterly minimum + signal-driven touch-ups.
- **Cards rotting without triggers:** If no score-10+ signals in a quarter, still do a quarterly review.
- **One person's knowledge:** If only one person knows competitor X, the system is fragile. Document.
