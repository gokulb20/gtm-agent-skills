# Cost Calculation — Channel Performance

## Channel Cost Formula

```
channel_cost = tool_subscription + per_touch_credits + (rep_hours × rep_loaded_cost_per_hour × allocated_share)
```

## Component Breakdown

| Component | Source | Notes |
|---|---|---|
| Tool subscription | [user-provided] | Smartlead, HeyReach, JustCall, etc. |
| Per-touch credits | [user-provided] | Credit cost per email/LI touch/call |
| Rep hours | Estimated from activity volume | Time per touch × volume |
| Rep loaded cost/hour | [user-provided] | $80–150/hr loaded (salary + benefits + overhead) |
| Allocated share | Proportional | Share of rep's time on this channel vs others |

## Shared Cost Allocation

Shared costs (sender pool maintenance, infrastructure amortization) allocated proportionally to each channel's touch volume.

## Key Insight

A channel with no tool subscription (cold call) but heavy rep time is often more expensive per meeting than a channel with high tool spend but automated touches. Always include rep time.
