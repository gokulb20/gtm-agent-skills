# Report Template — KPI Reporting

## One-Screen Report Structure (≤450 words)

### North-Star Section (2 lines)
- Metric value + WoW % + 4-week trend direction
- Example: "$42K pipeline added (+18% WoW, ↑ vs 4w avg)"

### Leading Metrics Table (compact)
| Metric | This week | WoW abs | WoW % | Trend | Benchmark |
|---|---|---|---|---|---|
| Email touches | 800 | +160 | +20% | ↑ | baseline |
| LI connects | 60 | -5 | -8% | → | 12% floor |
| Calls made | 50 | -10 | -17% | ↓ | 8% target |
| Replies | 32 | +2 | +7% | → | 3% |
| Meetings | 14 | +3 | +27% | ↑ | — |

### Lagging Metrics Table (compact)
| Metric | This week | WoW | Trend |
|---|---|---|---|
| Closed-won | $34K (3 deals) | +$12K | ↑ |
| Tier-1 win rate | 12% | -6pp | ↓⚠️ |
| Cycle time D→P | 32d | +4d | ↑⚠️ |

### What's Working (2–3 items)
1. Email copy quality held at +20% volume — reply rate stayed at baseline
2. LI Vision-opener variant 18% acceptance — well above 12% floor

### What's Not (2–3 items)
1. Call connect 4% vs 8% target → re-verify phones via data-enrichment
2. Tier-1 win rate dropped 6pp → monitor; if persists 1 more week trigger icp-refinement-loop

### Recommended Actions
1. Run data-enrichment phone re-verify for active call list
2. Monitor Tier-1 win rate — threshold: 1 more week below 15% → trigger icp-refinement-loop

## Deeper Data (linked, not embedded)
- Per-channel detail → channel-performance report
- Pipeline detail → pipeline-stages audit
- Forecast detail → revenue-forecasting scenario breakdown
